# Installation

# Repository Clone

```shell
$ git clone https://github.com/piso-deepdive-project/DUI.git
$ npm i
```

# Activate Server and Build

```shell
$ npm start
```

# 프로젝트에서 집중하고자 했던 점

- 요구사항을 분석한다.
- 요구사항에 맞는 기능을 구현한다.
- 가독성을 스스로 검사한다.
- 코드리뷰를 통해 코드를 리팩토링한다.
- 추가로 필요로한 요구사항이 있는지 분석한다.
- 반복한다.

## 코드리뷰 체크리스트

- [ ] 코드가 읽기 좋은가? 가독성이 있는가?
- [ ] 불필요한 동작이 반복되지 않는가?
- [ ] 기본적인 구조를 꼭 유지해야 하는가?
- [ ] 변수명은 적절한가?
- [ ] 버그가 있진 않은가?
- [ ] 기능은 잘 동작하는가?
- [ ] 하나의 메서드가 너무 많은 일을 하진 않는가?
